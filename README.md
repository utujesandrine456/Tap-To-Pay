📱 Tap-To-Pay Project

Tap-To-Pay is a smart digital payment system that allows users to perform quick and secure payments by tapping RFID cards on a reader connected to an IoT device. The system communicates with a backend server using MQTT messaging and updates transaction balances in real time. It is designed to be fast, secure, and suitable for modern cashless payment environments.

🚀 Project Features

✅ RFID card detection and reading ✅ Real-time balance synchronization ✅ MQTT-based communication between device and server ✅ Secure transaction status updates ✅ Web dashboard for monitoring payments

The Tap-To-Pay system aims to provide a simple and efficient cashless payment solution suitable for small businesses, campuses, and transport payment systems.
